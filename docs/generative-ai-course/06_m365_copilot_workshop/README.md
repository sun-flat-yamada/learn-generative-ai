# 第6章: Microsoft 365 Copilot 活用ワークショップ

> **形式:** ハンズオン（個人またはペアワーク）  
> **所要時間:** 約90分  
> **準備:** Microsoft 365 Copilot ライセンス、Word/Excel/Teams/Outlook アクセス

---

## M365 Copilot の特徴おさらい (2026年版)

```
Work IQ & Agentic AI（組織知と自律実行）
    ↓
Microsoft 365 Copilot (Wave 3)
    ├── Copilot Pages（AIとの共同キャンバス）
    ├── Word（文書作成・高度な参照・Edit with Copilot）
    ├── Excel（Python連携・Agent Modeによる多段階分析）
    ├── PowerPoint（ナラティブ生成・動的アニメーション）
    ├── Outlook（スマート要約・自律型スケジュール調整・音声 triage）
    ├── Teams（高度な会議エージェント・多言語リアルタイム対応）
    └── Copilot Chat（GPT-5 / Claude 連携、SharePoint 全文検索）

セキュリティ & モデル:
✓ データはMicrosoft 365テナント内で保護（外部漏洩なし）
✓ 学習に利用されない（商用プライバシー保護）
✓ 最新モデル（GPT-5.4 / Claude 3.7+）の選択が可能
✓ Role-based Access Control (RBAC) の自動継承
```

---

## 2026年の注目アップデート：Agentic AI と Copilot Pages
- **Agentic AI**: 単なる回答だけでなく、会議のスケジュール調整やデータの整理など、複数のステップを伴う「タスク」を自動実行します。
- **Copilot Pages**: AIとの対話から生まれたアイデアを、即座にWebページのようなキャンバスへ「固定」し、チームで共同編集できる新しい働き方です。

## お題1: Teams 会議でのCopilot活用

### シナリオ

```markdown
会議設定:
- タイトル: 2026年度Q1 事業計画検討会議
- 参加者: 田中部長、山田課長、佐藤（あなた）、他3名
- 会議時間: 60分
- 会議ファイル: src/workshops/m365_copilot/meeting_agenda.md を使用
```

### ステップ1: Teams で Copilot を有効にする

```markdown
1. Teams の会議開始時に「Copilot」ボタンをクリック
2. 「今すぐ有効にする」を選択（主催者のみ）
3. 会議中にCopilotパネルが表示される
```

### ステップ2: 会議中のリアルタイム活用（多言語・エージェント対応）

```markdown
使えるコマンド例:
- 「ここまでの議論を要約して」
- 「意見が分かれているポイントを整理して」
- 「田中部長（日本語）の発言を英語でリアルタイム要約して」
- 「まだ話し合われていない議題は？」
```

### ステップ3: 会議後の「自律型」アクション生成

```markdown
会議終了後にCopilotが提案するもの:
- 議事録（主要な議論のポイント）
- アクションアイテム（自律エージェントによるTo-Do登録の提案）
- 関連ドキュメントの自動検索・リンク
```

**チャレンジプロンプト（Agentic Command）:**

```markdown
会議後のCopilot Chatに入力:
「Q1事業計画会議の内容に基づき、以下のタスクを実行して:
1. 決定事項をOutlookの予定表に『Q1マイルストーン』として登録して
2. 担当者の山田さんに、アクションアイテムをTeamsで個別に通知して
3. 会議の重要ポイントを反映した、田中部長へ送るための週次レポート（Word）のドラフトを作成して」
```

## お題2: Excel Copilot でデータ分析（Python & Agent Mode）

> 📁 データファイル: `src/datasets/sales_data.csv` を使用してください

### ステップ1: Excelでファイルを開き、Agent Modeを起動

```markdown
1. sales_data.csv を Excel で開く（Ctrl+T でテーブル化）
2. リボンの「Copilot」ボタンをクリック
3. 「Agent Modeをオンにする」を選択（複雑な分析・Python実行が可能になります）
```

### ステップ2: Pythonを活用した高度な分析

**2026年の新機能：Python in Excel の実行**

```markdown
「このデータを使って、翌四半期の売上予測をProphetモデル（Python）で作成して。結果を新しいシートにグラフ付きで出力して」
「散布図行列を作成して、製品カテゴリ別の相関関係をPythonで分析して」
```

### ステップ3: 自然言語によるマルチステップ分析

**試してみるプロンプト集:**

```markdown
基本分析:
「月別売上の傾向を教えて」
「最も売れた製品TOP5を表示して」

Agentic分析:
「売上が低い原因を特定し、改善のための具体的なシミュレーションを作成して」
「在庫状況と連携して、Q2の発注推奨リストを作成して」
```

### ステップ4: 数式の自動生成

```markdown
入力例:
「各月の売上目標（前月比110%）を計算する数式を追加して」
「年間累計売上を表示するDAX/SUMIF数式を書いて」
```

## お題3: Word Copilot でドキュメント作成

> 📁 テンプレート: `src/workshops/m365_copilot/report_template.docx` を使用

### シナリオ

```markdown
タスク:
先週のQ1業績レポートを作成する。
データは sales_data.csv、meeting_notes.txt、および共有フォルダのPDF資料を参照すること。
```

### ステップ1: 白紙から作成（マルチソース参照）

```markdown
Wordを開き、Copilotに入力:
「Q1売上業績レポートを作成してください。 [sales_data.csv] と [meeting_notes.txt] を参照。
含める内容:
- エグゼクティブサマリー
- 主要KPIのハイライト
- 課題と改善策（2025年の市場動向PDFとの比較を含めて）

トーン: 経営幹部向け、簡潔かつ具体的」
```

### ステップ2: 文中でAIと直接対話（Edit with Copilot）

```markdown
既存の文章を選択し、ポップアップの「Edit with Copilot」をクリック:
- 「このセクションを、よりデータに基づいた説得力のある内容に書き換えて」
- 「専門用語を経営者でも理解できる言葉に置き換えて」
- 「全体を通して論理的な一貫性があるかチェックして」
```

## 💡 TIPS: M365 Copilot の「進化する活用法」

> **Copilot Studio & SharePoint Grounding:**
>
> 「社内の出張旅費規程（SharePoint上のPDF）に基づいて、今回の出張申請の不備を指摘して」
> → 特定のデータソースを「グラウンディング（根拠付け）」に使うことで、嘘（ハルシネーション）を激減させます。
>
> **Agentic Workflow:**
>
> 「未承認の経費精算をリストアップし、承認者にリマインドを送って」
> → 単なる情報抽出を超え、システムの操作をエージェントに任せることが2026年のスタンダードです。
>
