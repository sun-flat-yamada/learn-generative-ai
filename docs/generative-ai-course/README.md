# 生成AI学習コース (Generative AI Course)

本ディレクトリは、生成AIの基礎から実用的なスキル、そしてセキュリティまでを体系的に学習するためのコース資料を格納しています。
新入社員や生成AI初学者を対象とし、「なぜ」を理解し「どう使うか」を実践できる構成になっています。

## 📚 コース構成

コースは以下の7つの章で構成されています。

### 01. 生成AIの基礎 ([01_generative_ai_basics](./01_generative_ai_basics/))

生成AIとは何か、その歴史や仕組み、主要なモデルの比較など、土台となる知識を学びます。

- 生成AIの定義と歴史
- 仕組み（大規模言語モデルなど）
- 制約とリスク
- 用語集

### 02. 生成AIの使い方 ([02_how_to_use](./02_how_to_use/))

効果的なプロンプトの書き方（プロンプトエンジニアリング）や、各種ツールの使い分け、ナレッジの活用方法を学びます。

- プロンプトエンジニアリング基礎
- プロンプトクックブック（実用例）
- ツールの種類と使い分け
- MCP (Model Context Protocol) を活用した知識連携

### 03. セキュリティ ([03_security](./03_security/))

企業で生成AIを安全に利用するために不可欠な、リスクへの理解と対策について学びます。

- セキュリティリスクの把握
- 具体的なセキュリティ対策

### 04. 継続的な学習 ([04_learning](./04_learning/))

日進月歩のAI技術を追い続けるための方法や、役立つ情報ソースを紹介します。

- 継続的学習の重要性
- 役立つリンク集

### 05. ChatGPT ワークショップ ([05_chatgpt_workshop](./05_chatgpt_workshop/))

ChatGPTを用いた実践的な演習を通じて、基本操作から高度な活用までを体験します。

### 06. Microsoft 365 Copilot ワークショップ ([06_m365_copilot_workshop](./06_m365_copilot_workshop/))

業務で利用するOfficeツール内でのCopilot活用方法を実践的に学びます。

### 07. GitHub Copilot ワークショップ ([07_github_copilot_workshop](./07_github_copilot_workshop/))

エンジニア向けのAI支援ツールであるGitHub Copilotを用いた開発効率の向上を体験します。

---

## 🚀 学習の進め方

1. `01` から順番に読み進めることを推奨します。
2. 座学（01-04）で理論を理解した後、ワークショップ（05-07）で実際に手を動かしてください。
3. わからない用語があれば `01_generative_ai_basics/07_terminology.md` を参照してください。
