# Workshop Bundle: Generative AI Basics

This directory contains resources for the "Generative AI Basics" workshop.

## Contents

1. `sample_texts.json`: Example text datasets for summarization, sentiment analysis, and translation tasks.
2. `prompt_templates/`: Reusable prompt templates for common business scenarios.
