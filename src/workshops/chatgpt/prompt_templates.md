# ChatGPT ワークショップ サンプルプロンプト集

このファイルには、お題で使えるプロンプトテンプレートをまとめています。
そのままコピーして ChatGPT に貼り付けて使ってください。

---

## お題1: 業務メール用プロンプトテンプレート

### 基本テンプレート

```
# Role（役割）
あなたは[役職]として[会社]に勤めています。

# Context（背景）
[状況の説明]

# Task（タスク）
[何をしてほしいか]

# 要件
- [要件1]
- [要件2]
- [要件3]

# Format（形式）
- 件名あり
- 丁寧なビジネスメール形式
- [文字数]以内
- 署名は「[お名前]」
```

### フォローアップメール用

```
# Role（役割）
あなたはIT企業の営業担当（経験5年）として勤務しています。

# Context（背景）
先週、{会社名}の{担当者名}{役職}に{製品名}のデモを実施しました。
デモは技術面では好評でしたが、担当者から「{懸念点}」という
反応がありました。

# Task（タスク）
フォローアップメールを作成してください。

# 要件
- 先週のデモへの感謝を伝える
- {懸念点}に対する理解を示す
- {次のアクション}を提案する
- {付加価値}を示唆するメッセージを含める
- 返信を促す質問で締める

# Format（形式）
- 件名あり
- 本文は丁寧なビジネスメール形式
- 文字数: 400字以内
- 署名は「[お名前]」と記載
```

## お題2: データ分析プロンプト

### CSVデータ分析テンプレート

```
以下のCSVデータを分析してください。

```csv
[CSVデータをここに貼り付け]
```

分析内容:
1. {分析項目1}
2. {分析項目2}
3. {分析項目3}

出力形式:
- 数値は表形式で示す
- 主要な発見事項を箇条書きで3点まとめる
- 改善提案を2点以上挙げる
```

### Python分析コード生成テンプレート

```
以下のデータを分析する Python コードを書いてください。

データの説明:
{データの説明}

必要な分析:
1. {分析1}（使用ライブラリ: pandas）
2. {分析2}（使用ライブラリ: matplotlib）
3. {分析3}

コード要件:
- Python 3.12 対応
- エラーハンドリングを含む
- 型ヒントを使用
- コメントを日本語で記載
- 関数に docstring を追加
```

## お題3: 意思決定分析プロンプト

### 多面的分析テンプレート

```
以下のビジネス意思決定について多面的に分析してください。

# 状況
{意思決定の背景と状況}

# 選択肢
## 選択肢A: {名称}
{詳細}

## 選択肢B: {名称}
{詳細}

# 分析ステップ
Step 1: 財務分析
- {財務指標1}を計算する
- {財務指標2}を計算する

Step 2: リスク分析
- 各選択肢のリスク要因を列挙する
- 影響度×発生確率でマトリクスを作成する

Step 3: 戦略的適合性
- {戦略目標}との整合性を評価する

Step 4: 最終推奨
- スコアリング（各項目5点満点）
- 推奨案と根拠
- 重要な注意点を3点

# 出力形式
- 各ステップを明確に見出しで区切る
- 数値は表形式で示す
- 最終推奨は「推奨: 選択肢〇〇」と明確に述べる
```

---

*これらのプロンプトは、目的に合わせて `{}` 内を書き換えてください。*
