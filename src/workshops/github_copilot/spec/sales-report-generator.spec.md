# 機能仕様: 売上レポート生成ツール (Sales Report Generator)

## 概要
社内の営業データ（CSV等）を読み込み、指定された月の売上レポートをMarkdown形式で自動生成するコマンドラインツール。

## [Role] 営業担当者が月次の成果を迅速に把握するためのレポート作成支援。
## [Mechanism] 
1. `src/datasets/sales_data.csv` を読み込む。
2. 指定された `YYYY-MM` に合致するデータを抽出。
3. 製品ごとの売上合計、地域別の売上比率を計算。
4. Markdown形式のレポートファイルを出力。

## 入力
- 対象月: `YYYY-MM` 形式（コマンドライン引数またはプロンプト）
- 出力先: ファイルパス（デフォルトは `report_{YYYY-MM}.md`）

## 出力形式
```markdown
# 月次売上レポート: {YYYY年MM月}

## サマリー
- 総売上: {合計値} 円
- 総販売数: {合計個数} 個

## 製品別売上（上位5件）
| 製品名 | 売上 | 販売数 |
| :--- | :--- | :--- |
| {製品A} | {金額} | {個数} |

## 地域別パフォーマンス
- East: {比率}%
- West: {比率}%
...
```

## エラー処理
- 対象月のデータが存在しない場合: 「指定された月のデータが見つかりません」と表示。
- CSVファイルが読み込めない場合: エラーの内容を表示して終了。

## テスト要件
- 正常系: 正しい計算結果がMarkdownに含まれていること。
- 異常系: 不正な日付形式の入力に対するバリデーション。
